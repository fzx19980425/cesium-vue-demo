
# 以美国先锋1号的tle数据为基准，自定义了tle数据，并生成对应的卫星图

```json
{
    "satellite_101": {
    "tle": [
      "1 00018U 59005  A 59269.32477950  .04655228 +00000-0 +00000-0 0  9998",
      "2 00018 089.9899 201.4819 0140209 012.7649 347.4143 14.48843120006800"
    ],
    "country": "CN",
    "role": "LEO",
    "networkSystem": "",
    "type": "",
    "networkState": "",
    "toSatellitePolyLines": [],
    "toRadarPolyLines": []
  },
}
```

## satellite_101是卫星名称

## tle是卫星的tle数据

### 数据格式如下

第一行：
行号：1
卫星编号：00018U
国际设计ator年份的最后两位：59
发射年份的日子：005
发射时刻的小数部分（UTC 时间）：A 59269.32477950
卫星的第一次导数修正，表示每天平均运动的变化率。通常以圈/日^2 为单位：.04655228
卫星的第二次导数修正，表示每天平均运动变化率的变化率。通常以圈/日^3 为单位：+00000-0
卫星的 BSTAR drag term，用于考虑大气阻力对卫星轨道的影响。这个值越大，表示大气阻力对轨道的影响越明显：+00000-0
卫星的 Ephemeris type，指示使用的模型类型:0
校验和，用于验证 TLE 数据的正确性:9998
第二行：
行号：2
卫星编号：00018
轨道倾角,卫星轨道平面与地球赤道平面之间的夹角：89.9899
升交点赤经，卫星轨道平面与参考平面（通常是地球赤道平面）的交点在参考平面上的经度：201.4819
轨道偏心率,描述了轨道的椭圆程度。增加偏心率会使轨道更加椭圆：0140209
近地点幅角，升交点到近地点的轨道段在轨道平面上的角度，沿着卫星轨道的方向测量：012.7649
平近点角，卫星在轨道上所处位置与近地点之间的角度：347.4143
平均运动（每日）表示卫星每天绕地球公转的平均圈数：14.48843120006800 圈/日

## country是卫星所属的国家

## role是卫星的角色

## networkSystem是卫星所属的系统

## type是卫星的类型

## networkState是卫星的网络状态

## toSatellitePolyLines是卫星之间的连线

## toRadarPolyLines是卫星与雷达之间的连线
